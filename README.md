## <span style="color:#7EE787">📌 Descripción</span>

**Rick and Morty App** es una aplicación web full stack desarrollada con **React** y **Node.js**.

Permite buscar personajes de **Rick and Morty** por ID, generar personajes aleatorios, ver información detallada y guardar personajes favoritos.

El frontend está desarrollado con React y Redux, mientras que el backend está construido con Express y consume datos de la API pública de Rick and Morty.

## <span style="color:#A371F7">✨ Funcionalidades</span>

- Buscar personajes por ID.
- Generar personajes aleatorios.
- Ver detalle individual de cada personaje.
- Agregar y quitar personajes de favoritos.
- Eliminar personajes del Home sin quitarlos de favoritos.
- Navegar entre Home, About, Favorites y Detail.
- Login con usuario demo.

## <span style="color:#FF7B72">⚙️ Instalación local</span>

Clonar el repositorio:

```bash
git clone https://github.com/MarianoMasondo/rick_and_morty_marian.git
cd rick_and_morty_marian
```

Instalar y ejecutar el backend:

```bash
cd server
npm install
npm start
```

Instalar y ejecutar el frontend:

```bash
cd ../client
npm install
npm start
```
