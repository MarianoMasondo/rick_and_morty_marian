# Rick and Morty App

Aplicación web full stack para buscar personajes de **Rick and Morty**, ver su detalle y guardar favoritos.

## Demo

Frontend: https://rick-and-morty-marian.vercel.app/

Backend: https://name-rick-and-morty-marian-api.onrender.com/rickandmorty/character/1

## Usuario demo

```txt

```

## Funcionalidades

- Buscar personajes por ID.
- Generar personajes aleatorios.
- Ver detalle de cada personaje.
- Agregar y quitar favoritos.
- Eliminar personajes del Home sin quitarlos de favoritos.
- Navegar entre Home, About, Favorites y Detail.
- Login con usuario demo.

## Tecnologías

**Frontend:** React, React Router, Redux, Axios, CSS Modules.  
**Backend:** Node.js, Express, Axios, CORS.  
**Deploy:** Vercel y Render.

## Instalación local

Clonar el repositorio:

```bash
git clone https://github.com/MarianoMasondo/rick_and_morty_marian.git
cd rick_and_morty_marian
```

Instalar y ejecutar el backend:

```bash
cd server
npm install
npm start
```

Instalar y ejecutar el frontend:

```bash
cd ../client
npm install
npm start
```
